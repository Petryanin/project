## Setup
- install and run [docker](https://www.docker.com/)
- open CLI and run these commands
```
git clone https://github.com/Petryanin/project.git
cd project
docker-compose up
```
- use this [link](http://127.0.0.1/) or type **127.0.0.1** in browser address bar
